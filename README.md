# lada-pm.com

Personal website for **Lada Borovikova**, Ph.D. — Generative AI Leader and AI Product Manager.

## Structure

```
lada-pm/
├── website/          # Static website files
│   ├── index.html    # Home page
│   ├── about.html    # About page
│   ├── portfolio.html# Portfolio page
│   ├── styles.css    # Shared styles
│   ├── images/       # Images (logo, profile, background)
│   └── game/         # AI terminology game
├── server.py         # Local development server
└── prompts/          # Build history / prompts log
```

## Pages

- **Home** — Introduction and key areas of expertise
- **About** — Background, journey, and LinkedIn profile
- **Portfolio** — Projects and accomplishments
- **AI Game** — Interactive word game for AI and statistics terminology

## Running Locally

```bash
python3 server.py
```

Opens at [http://localhost:3000](http://localhost:3000)

## Tech

Static HTML/CSS/JS site with a responsive hamburger navigation menu and a blue gradient color theme.
